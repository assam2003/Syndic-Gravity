# 🏢 Syndic Gravity

A premium, modern property management solution designed for syndics and property owners to manage residents, finances, and maintenance with ease.

![GitHub last commit](https://img.shields.io/github/last-commit/assam2003/Syndic-Gravity?style=for-the-badge&color=blue)

---

## ✨ Features

- **📊 Dashboard Overview**: Real-time insights into property health and finances.
- **👥 Resident Management**: Complete database of residents with contact info and unit details.
- **💰 Financial Tracking**:
  - **Revenue**: Track monthly contributions (Cotisations) and annual income.
  - **Expenses**: Detailed logging of maintenance and operational costs.
  - **Repair Funds**: Dedicated tracking for property improvement projects.
- **📂 Document Generation**: Generate professional reports for meetings and audits.
- **🔒 Secure Access**: Role-based login system to protect sensitive data.

---

## 🚀 Tech Stack

- **Frontend**: HTML5, CSS3 (Modern Glassmorphism UI)
- **Logic**: Vanilla JavaScript
- **Styling**: Custom CSS Systems (Theme-based)

## 🛠️ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/assam2003/Syndic-Gravity.git
   ```
2. **Open index.html**
   Simply open the `index.html` file in any modern web browser to start using the application locally.
